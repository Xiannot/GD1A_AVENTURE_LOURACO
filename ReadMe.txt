Utilisation des touches : 

Z : Avance
Q : Gauche
D : Droite
S : Reculer
1 : Changement d'arme
2 : Changement d'arme
3 : Consomé l'item
N : Apparaitre curseur
B : Disparaitre Curseur 
H : Pour se give du Heal

La manette est disponible partout sauf dans le menu ou l'utilisation de la sourit se fait obligatoire.
De nombreux bugs sont encore disponible du au manque de temps Merci de votre compréhension .

Touche Manette:

Joystick gauche : déplacement
Joystick Droit : visé

A : Ramasser 
X : Consumé l'item

Right Shoulder : Attaqué
RB/LB : Changement d'arme

L'ors du démarage du jeux veuiller suivre les Panneau dans l'ordre suivant : N1 - N2 - N3
Pour suivre l'ordre de difficulté.

Le jeux se termine a la fin des 3 Niveau il n'y a pas d'écran finale.

Aucun écran de mort est disponible pour l'instant l'orsque vous avez l'animation de cendre veuillez redamaré le jeux
